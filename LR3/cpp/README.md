# Orders system for C++(20)

## Структура и задание 

Система поделена на следующие модули:
- `infrastructure.h`: Имитация работы с БД и внешними сервисами.
- `main.cpp`: Точка входа в приложение.
- `models.h`: Модели данных системы.
- `processor.h`: Основная бизнес-логика.
- `staff.h`: Система управления персоналом склада.

Почти в каждом модуле есть архитектурные ошибки, которые нужно исправить.

## Использование и зависимости
Проект требует CMake для сборки и C++20 для корректной работы

Запуск системы: 
```
cmake --build
./orders_c++
```
Или при помощи контекстуального меню в VScode
